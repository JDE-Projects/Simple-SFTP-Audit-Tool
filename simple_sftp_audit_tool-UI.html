Run DoS resilience check?

When on, the audit opens dozens of rapid connections to test whether the server is vulnerable to the DHEat denial-of-service attack (CVE-2002-20001). This can look like an attack: it may trigger rate-limiting, fail2ban, or intrusion detection and temporarily block your IP, generate many log entries, and slow the audit. Only enable it for servers you own or are authorized to test this way.
